# Sample Leases Directory

This directory contains sample commercial real estate lease documents for testing and demonstration purposes.

## Expected Contents

Place your PDF lease documents here:

- `base_lease_001.pdf` - Sample base lease agreement
- `base_lease_002.pdf` - Another sample base lease
- `amendment_001.pdf` - Sample first amendment
- `amendment_002.pdf` - Sample second amendment
- `sublease_001.pdf` - Sample sublease agreement

## Document Requirements

For best results, documents should:

1. Be text-based PDFs (not scanned images)
2. Follow standard commercial lease formatting
3. Be in English
4. Be under 50MB in size

## Privacy Notice

**Do not commit real lease documents containing sensitive information to version control.**

Use redacted or synthetic documents for testing.
